# ☎️ Java Contact Manager

A simple **command-line contact manager** built in **Java** using **SQLite**.

## 🚀 Features
- ✅ **Add contacts** (Name, Phone Number)
- 📋 **View all contacts**
- ✏️ **Update contact information**
- 🗑️ **Delete contacts**
- 💾 **Data persistence** using SQLite

## 📌 Technologies Used
- **Java** (Programming Language)
- **SQLite** (Database)
